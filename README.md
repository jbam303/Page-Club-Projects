# Code Club - Landing Page

Esta es la página de aterrizaje (landing page) oficial del **Code Club**, un espacio dedicado a la comunidad, donde el código cobra vida. El objetivo principal de este proyecto es presentar el club, mostrar nuestra misión y visión, así como atraer a nuevos miembros interesados en aprender, colaborar y crear proyectos reales.

## Características de la Plataforma

- **Página Principal (Landing):** Presenta la propuesta de valor del club, nuestros principios (crecimiento profesional, aprendizaje a través del error), nuestros horarios de disponibilidad, y nuestras metas para el 2026 (Hackathon Anual, Vinculación con el medio, Fondo concursable).
- **Página de Registro:** Permite a los nuevos usuarios registrarse en la comunidad proporcionando su nombre, correo institucional, biografía y áreas de interés (Frontend, Backend, Videojuegos, Ciberseguridad, IA).
- **Diseño Moderno:** Utiliza un estilo futurista, tipo consola/terminal con colores neón vibrantes, efectos de "resplandor" y micro-interacciones.

## Tecnologías Utilizadas

- **React + Vite:** Para una experiencia de desarrollo rápida y un empaquetado optimizado.
- **Tailwind CSS:** Para el diseño de interfaz ágil, moderno y responsivo.
- **React Router:** Para la navegación y enrutamiento ágil entre componentes de la aplicación.

## Cómo Ejecutar el Proyecto

1. Clona el repositorio y ve al directorio del proyecto:
   ```bash
   cd Page-Club-Projects
   ```

2. Instala las dependencias necesarias:
   ```bash
   npm install
   ```

3. Ejecuta el servidor de desarrollo local:
   ```bash
   npm run dev
   ```

4. Abre tu navegador en la URL que indique la consola (usualmente `http://localhost:5173/`).
